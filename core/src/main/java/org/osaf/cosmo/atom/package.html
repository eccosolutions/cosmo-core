Implementation for Cosmo's atom-based protocol for retrieving/updating user data.

Package Specification

Atom classes are roughly broken down into three categories:

The generator subpackages contain classes responsible for generating atom feeds.

The processor package contains classes responsible for processing data included in atom requests.

The provider package contain classes that implement the atompub protocol interfaces from abdera.
